# 1. 오작교에 관하여
## 1.1. 오작교란?
오작교(烏鵲橋)는 까마귀와 까치가 만든 다리라는 뜻으로,  
애인 관계인 사람 혹은 친구와 친구 애인 사이를 더 돈독하게 만드는 사람을 비유할 때 쓴다.

## 1.2. <오작교>는 어떤 서비스인가?
[**6단계 분리 이론**](https://www.youtube.com/watch?v=GNzeH9HMkYY)에 따르면 인간관계는 **6단계**만 거치면 지구상 대부분의 사람과 연결될 수 있다.  
<오작교>는 관심있는 혹은 소개받고 싶은 사람과 연결되려면
누구를 거쳐야하는지 알려줄뿐더러 연결되도록 도와주는 서비스이다.
